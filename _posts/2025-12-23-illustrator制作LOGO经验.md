### 思路 & 原理

- fill 和 stroke，钢笔和画笔：

  | 工具                | 本质     | SVG 结果    | 能否手写动画 |
  | ------------------- | -------- | ----------- | ------------ |
  | **钢笔（Pen）**     | 几何路径 | stroke path | ✅ 可以       |
  | **铅笔（Pencil）**  | 几何路径 | stroke path | ✅ 可以       |
  | ❌ **画笔（Brush）** | 外观效果 | fill 轮廓   | ❌ 不行       |

- SVG - M（move）/L（画线）/C（曲线）/Z（闭合）。在内联 `<style>` 中进行查看。

  - VS Code 中 SVG 图片转代码方法：右键「打开方式」--> 「文本编辑器」

- 钢笔描制**骨架**（用于动画）（先普通钢笔定锚点，接着曲率等工具来圆滑文字边缘） + 描边外扩加外观（路径的「描边」选项中，加粗描边；将端点、边角改成圆角）

  - 「长按 ctrl + 钢笔」：点击、选择单个锚点
  - ……（还没研究明白）
